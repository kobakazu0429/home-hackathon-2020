# home-hackathon-2020

## How to dev

### Setup

```bash
# ... Setup node and yarn
$ git clone <git repository url>
$ cd <your project name>
$ yarn install
```

### Commands

```bash
# written in package.json
$ yarn dev   # Start webpack-dev-server
$ yarn build # Build to dist/
$ yarn lint  # Check syntax
$ yarn typecheck  # Check type
```
